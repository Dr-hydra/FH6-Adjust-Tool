# FH6 调校工具 (FH6 Adjust Tool)

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![WPF](https://img.shields.io/badge/platform-WPF%20%7C%20.NET%2010.0-orange.svg)

`FH6 调校工具` 是一款专为《极限竞速：地平线 6 (Forza Horizon 6)》打造的第三方开源车辆物理调校与 AI 智能调优助手。

它融合了地平线社区成熟的物理调校算法，并搭载了先进的 AI 大语言模型（支持多供应商 API 接入），为竞速游戏爱好者提供一站式的车辆属性计算、方案存档及人机交互式的连续微调指导。

---

## 🌟 核心功能

1. **精准物理计算**
   - 车型规格直接读取 `fh6-file-analyse` 的 FH6 游戏 SQLite 数据库，可自动载入原厂车重、前部配重、PI、驱动形式、轮胎、转速、扭矩、极速、终传和逐挡齿比，不再依赖 `cars.json`。
   - 针对不同驾驶风格（**抓地/漂移/直线加速/拉力离地**）自动推荐最优的胎压、避震弹簧刚度、阻尼常数、防倾杆刚度以及齿轮传动比。
   - 支持调校结果的**手动编辑与微调**，完美兼顾理论计算与个性化驾驶习惯。

2. **AI 智能诊断与连续对话**
   - **多供应商 API 兼容**：原生支持官方 **Google Gemini** 接口，并全面兼容 **OpenAI 格式标准**（可无缝接入 DeepSeek、Claude、Ollama 本地模型或国内各类 API 代理）。
   - **一键增强调校**：AI 将直接读取当前车辆的属性及计算出的物理调校结果，进行多维度的赛道表现诊断并输出优化报告。
   - **连续聊天问答**：调优完成后，左侧栏会自动展开连续对话聊天室。用户可就车辆表现（如“高速弯推头”、“起步打滑”）向 AI 调校助手追问，AI 将结合历史记录与当前车辆配置给予精细的修改方向。
   - **模型自定义**：设置页面支持自主选择或输入自定义模型名称、接口地址（Base URL）及 Key。

3. **本地方案存档与分享**
   - 调校方案支持重命名、另存为及彻底覆盖/删除。
   - 内置**调校分享码**功能。支持将调校一键生成紧凑的分享码发给好友，或通过输入分享码快速导入他人的调校配置。
   - 支持只读扫描本地 FH6 存档中的 `Tuning_*` 调校容器，提取作者、调校名、车型、升级件和调校滑块，按车辆与已安装零件的物理范围还原 UI 实际数值，勾选后选择性导入为本地方案。
   - 导入时根据升级件推导改装重量、配重、驱动形式、轮胎尺寸、挡位数和红线转速；无法从调教文件可靠恢复的最终 PI、扭矩和极速会明确标记为原厂参考或未知。
   - 根据已安装轮胎推荐 Race、Drift、Drag、Rally 等计算模式和路面；导入候选及方案详情会按分类展示非原厂配件摘要，包括发动机互换与发动机内部升级件。
   - 设置页可手动指定存档目录；留空时会优先按 `C:\XboxGames\GameSave\pgs` / 各盘符 `XboxGames\GameSave\pgs` 自动识别。

4. **遥测仪表盘与圈速分析**
   - 支持监听游戏 **Data Out UDP** 数据流，实时显示速度、转速、档位、油门、刹车、当前圈计时与当前调校上下文。
   - 内置 WebView2 遥测仪表盘，可绘制路线轨迹，并支持对最近样本进行回放查看。
   - 支持手动标记起终点，用于 Time Attack 路线圈速统计；同时可读取 Rivals/计时模式中的游戏圈速数据。
   - 自动记录圈速、车辆、PI、驱动形式和调校名称，并提供**调校对比**视图，帮助评估不同调校方案的最佳圈速与平均表现。
   - 支持自动和手动问题标记，例如抓地极限、悬挂压缩到底、重刹锁死等，用于复盘调校弱点。
   - 遥测样本、圈速、会话和问题标记默认写入本地 `telemetry.db`，便于离线分析和后续复盘。

## 🆕 v3.1.1 更新内容

- **统一 FH6 游戏数据库**：车辆名称、原厂规格、传动系统和升级件全部改由上游 SQLite 数据库与车辆名称表提供，移除 `cars.json`；数据库文件支持随上游数据同步。
- **存档调校选择性导入**：扫描本地 `Tuning_*` 容器后默认全不选，支持车型筛选、勾选导入、来源去重和同来源方案原位刷新。
- **改装规格与配件解析**：按已安装升级件估算改装重量和前配重，并解析驱动互换、轮胎尺寸、挡位数、红线转速、发动机零件及其他非原厂配件摘要；最终 PI 等无法可靠推导的数据会明确标记来源。
- **实际 UI 调校数值**：根据升级件对应的动态范围还原弹簧、车高、防倾杆、阻尼和下压力，支持存档末尾第 10 挡半精度值解析。
- **方案数据库与批量管理**：保存方案改用 SQLite 按需加载，增加车型筛选、收起式详情、勾选批量删除、规格来源和配件分组展示，改善大量方案下的加载性能。
- **遥测与圈速分析**：提供 WebView2 实时仪表盘、路线轨迹、圈速记录、问题标记和调校对比，用于复盘不同方案的实际表现。
- **界面与导航修复**：二级导航与页面切换保持一致，方案卡片默认收起，调整调校模式图标，并隐藏无法可靠取得的风阻系数。

## 🛠️ 技术架构

- **UI 层 (WPF)**：基于 Windows Presentation Foundation 构建，自绘标题栏与高仿真自适应布局。
- **遥测仪表盘 (WebView2)**：以本地 Web 资源渲染实时仪表、路线画布、圈速表和调校对比表。
- **Core 核心库 (C# / VB.NET)**：
  - `TuningCalculator`：车辆物理学公式引擎，推导并输出基础悬挂和传动值。
  - `CarDatabase`：读取上游 FH6 游戏数据库和车辆名称表，并按文件版本在线同步。
  - `FH6TuneDatabase`：解析存档升级槽，估算改装重量并生成车辆/零件对应的动态 UI 范围。
  - `ShareCodec`：紧凑型 Base64 车辆规格与调校状态编解码器。
  - `AiClientFactory`：多 API 适配层，实现 `IAiClient` 以封装多供应商请求。
  - `TelemetryService`：Forza Data Out UDP 接收、样本解析、圈速识别与问题标记引擎。
  - `TelemetryStore`：基于 SQLite 的本地遥测会话、样本、圈速和调校对比存储。

## 🚀 编译与运行

### 环境依赖
- **操作系统**：Windows 7 SP1 或更高版本
- **开发工具链**：.NET 10.0 SDK

### 本地编译步骤
1. 克隆本项目仓库：
   ```bash
   git clone https://github.com/Dr-hydra/FH6-Adjust-Tool.git
   cd FH6-Adjust-Tool
   ```
2. 使用 .NET CLI 编译 Release 版本：
   ```powershell
   dotnet build .\FH6AdjustTool.sln -c Release
   ```
3. 编译完成后，即可在下方路径找到可执行文件并直接双击运行：
   `src\FH6AdjustTool\bin\Release\net10.0-windows\FH6AdjustTool.exe`

### 发布包说明

- `FH6AdjustTool_v3.1.1_FrameworkDependent.zip`：框架依赖单文件版，需要预先安装 .NET 10 Desktop Runtime (x64)。
- `FH6AdjustTool_v3.1.1_SelfContained.zip`：自包含单文件版，已内置 .NET 运行时，解压后可直接运行。
- 两种发布包都只保留主程序及必要的 `Data`、`Web` 资源；程序只从实际 EXE 目录和受控应用数据缓存加载车辆数据库，不会从系统临时目录搜索开发版解析器数据。
